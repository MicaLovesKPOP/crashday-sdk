Certain stunts and tricks in Stunt Mode are identified by the motion of a car, such as flips, rotations, jumps, good landings etc. They will work on any tile. However, all other stunts are tile-specific and thus needs to be defined per tile.

The stunt definitions for each track piece-.cfl are placed in a related .cst-file and set up using the StuntEdit, provided with this SDK. See StuntEdit for a documentation on how to add stunt information to your track pieces.
