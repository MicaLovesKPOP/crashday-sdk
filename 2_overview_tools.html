SDK tools directory


This is a short overview on all the tools delivered by the SDK.

MakeP3D

converts .3ds files into Crashdays .p3d format (see MakeP3D)

ZModeler p3d Export

plugin for Zanoza Modeler to export to p3d (see ZModeler export)

ShowP3D

p3d viewer that displays 3d models and shaders in the in-game engine (see ShowP3D)

ShowCFL

track piece viewer that displays track pieces using the in-game engine. It also displays AI information and vegetation (see ShowCFL)

SuntEdit

tool to define stunt tricks for particular actions you can perform on a track piece (see StuntEdit)

VegetEdit

tool to set and modify the appearance of vegetation on a track piece (see VegetEdit)

Car Manager

tool to import/export custom cars to/from Crashday (see Car Manager)
