After the Redline update all crashday files and mods are stored inside .cpk files. These files are just zip archives and contain all of the game's content. If you go to the crashday/data folder you will find all of the game's packs.

Do not edit any game's .cpk files in any way. Game's anticheat will not let you in the game if it finds any differences from the originals. If you want to change any content, use mods to do so.

.cdx files are encrypted packages and can not be opened.

This is a short overview on all the modding-relevant folders.

ambience

Contains ambience definition files. These are used to change weather and environment colors.

career

Contains all career missions definition files as well as ladder.lst containing enemies rankings

cars

Contains a subfolder for every car, contating different definition files

dbs

Contains dbs files which contain different game parameters

dynamics

Contains dynamic objects definitions

editor

Contains editor category definitions

fonts

Contains all the fonts, fonts sizes and conversion tables

loc

Contains localizations and credits file

models

Contains all the models

models/cars

Every subfolder contains a car's model

music

Contains music and in-game soundtracks

particles

Contains particle definiiton files

shop

Contains shop definitions files

shop/cars

Contains the weapons shop list. Every subfolder has it's corresponding car's shop list.

sounds

Contains all the sounds

textures

Contains all the textures

tiles

Contains definition files for track pieces (tiles) containing information such as tile dimensions, vegetation, AI helpers and links to the reference 3d models

vegetat

Contains vegetation definition files

wheels

Contains wheel definition files
